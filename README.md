# Future of Artificial Intelligence 🤖

A modern, interactive Next.js web application showcasing the potential and aesthetics of Artificial Intelligence and Neural Networks. The site features a premium, futuristic design with embedded 3D models and micro-animations.

## 🌟 Key Features

- **Interactive 3D Elements:** Integrated with `@splinetool/react-spline` to display a dynamic, interactive 3D robot/AI model right in the browser.
- **Focus Mode (Flashlight Effect):** An immersive visual mode that creates a "flashlight" beam following the user's cursor across the screen, mimicking localized lighting on the DOM.
- **Premium Glassmorphism UI:** Built with custom vanilla CSS focusing on blur effects, dynamic gradients, and modern typography.
- **Micro-Animations:** A rotating "System State" neural engine indicator giving the UI a constant feeling of analyzing and processing data.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** Custom Vanilla CSS for maximum flexibility and performance
- **3D Graphics:** [Spline Design](https://spline.design/) with `@splinetool/react-spline` + `three.js`

## 🚀 Getting Started

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/subhajitdas2005/Future-of-Artificial-intelligence.git
   cd "Future-of-Artificial-intelligence"
   ```

2. **Install dependencies:**
   Make sure you have [Node.js](https://nodejs.org/) installed, then run:
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the website:**
   Open [http://localhost:3000](http://localhost:3000) in your web browser.
